#### Username Signup Rules

The rules for allowable username characters are as thus:

1. Username must be between 8 and 20 characters.
2. The only characters allowed are letters, digits, underscore, and hyphens.
3. Username are not case sensitive, meaning they are saved as lowercase.
4. No whitespace, unicode, or special characters, like !@#

##### Additional Related Signup Rules

Further rules at Signup are as thus:

1. An email can only be registered once
2. An username can only be used once
3. Once an account is registered, the user must manually login
4. In order for a user to register, they must agree to the Terms of Service
5. In order for a user to register, they must agree to the Privacy Policy
